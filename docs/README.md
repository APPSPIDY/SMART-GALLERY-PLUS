# Smart Gallery+

**Keep memories. Lose clutter.**

Smart Gallery+ is an AI-powered gallery organizer that helps users rate photos into three categories:

### ⭐⭐⭐ Most Valuable

The moments that matter most.

### ⭐⭐ Worth Reviewing

Keep the best, clear the rest.

### ⭐ Ready to Clear

Safe to remove. Low regret.

## Features

* AI Rating
* User Rating
* Tags
* Notes
* Quick Scan
* Deep Scan
* Duplicate Detection
* Screenshot Detection

## Status

Currently in prototype stage.
